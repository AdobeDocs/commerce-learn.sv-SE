# Optimera Storefront med inbyggd experimentering

Upptäck hur Adobe Commerce as a Cloud Service gör det möjligt för affärsteam att köra A/B-tester och optimera butiksinnehållet utan att behöva förlita sig på utvecklare eller verktyg från tredje part. Lär dig hur du mäter påverkan på konverteringar och engagemang med inbyggda experimentfunktioner.

## Vem är den här videon till?

* E-commerce Manager
* Digital Marketing Specialist
* Konverteringsoptimeringsanalys

## Videoinnehåll

* Lär dig hur du konfigurerar och kör A/B-tester direkt i Adobe Commerce för att förbättra prestanda i butiken.
* Upptäck hur inbyggda experimenterings- och produktrekommendationer hjälper till att öka konverteringsgraden och engagemanget.
