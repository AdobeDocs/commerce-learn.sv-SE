# Utöka Adobe Commerce med App Builder &amp; API Mesh

Upptäck hur Adobe Commerce as a Cloud Service stöder utbyggbarhet via App Builder och API Mesh. Lär dig integrera backend-system, automatisera arbetsflöden och skapa anpassade appar utan att störa butiksupplevelsen.

## Vem är den här videon till?

* Lösningsarkitekt
* Backend Developer
* Integrationsspecialist

## Videoinnehåll

* Läs om hur App Builder möjliggör utveckling av anpassade appar och automatisering av arbetsflöden för Adobe Commerce.
* Upptäck hur API Mesh koordinerar data över flera system via ett enhetligt API-lager.
