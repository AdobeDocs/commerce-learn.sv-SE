# Funktionell demonstration för App Builder

Om du vill veta mer om hur du använder Adobe Commerce med App Builder kan du titta på videon nedan som visar hur du ansluter Adobe Commerce till Amazon Seller Central. Den här demonstrationen använder evenemang från Adobe Commerce som används av App Builder för att möjliggöra kontinuerlig integrering med Amazon Marketplace. Denna funktionsdemonstration beskriver hur denna integrering fungerar och utforskar möjligheterna i Adobe App Builder.
