# SQL REPORT BUILDER

Med Adobe Commerce Intelligence SQL Report Builder kan du fråga ditt datalager, granska resultaten och omvandla dem till diagram. När du skapar rapporter med SQL väntar du inte på uppdateringscykler för att förfina kolumnerna som du skapar. Om något ser ut kan du redigera och köra frågan igen tills utdata matchar det du förväntade dig.

>[!IMPORTANT]
>
> Du behöver administratörsbehörighet för att använda SQL Report Builder.

>[!TIP]
>
> Om du inte har använt Commerce Intelligence tidigare förklarar [Adobe Commerce Intelligence användarhandbok](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html?lang=sv-SE){target="_blank"} hur du samlar in, analyserar och visualiserar data för att få bättre affärsinsikter.

## Vem är den här videon till?

* Dataanalytiker

## Videoinnehåll

* Öppna SQL Report Builder och kör en fråga mot ditt datalager.
* Skapa en visualisering utifrån frågeresultaten.
* Spara rapporten på en kontrollpanel för ditt team.
