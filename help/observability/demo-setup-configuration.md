# Installation och konfiguration

Lär dig hur du får en total observerbarhet i alla Adobe Commerce-program och anpassade program med standardiserad, korrelerad telemetri. I den här videon förklaras varför korrelation och standardisering är viktigt. Här presenteras Commerce observationsmodul och IO Live Telemetry Library. Se hur dessa verktyg möjliggör enhetlig övervakning, loggning och spårning för ett smidigt operativt flöde.

## Vem är den här videon till?

* Adobe Commerce Developer
* Lösningsarkitekt
* Technical Product Manager

## Videoinnehåll

* Standardisera och korrelera telemetri för fullständig iakttagbarhet
* Använd Commerce observationsmodul för automatiska signaler
* Utöka övervakningen med IO Live Telemetry Library för appar
